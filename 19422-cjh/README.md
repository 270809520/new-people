[^1]: 人民币符号为¥。书写顺序为:先写大写字母"Y"，再在竖划上加上二横，即为"¥"，读音为:yuán(音:元)。在逐位填写金额的表格中用阿拉伯数字填写金额时，在金额首位之前加一个"¥"符号，既可防止在金额前填加数字，又可表明是人民币的金额数量。由于"¥"本身表示人民币的单位"元"，所以，凡是在金额前加了"¥ "符号的，金额后就不需要再加"元"字。
